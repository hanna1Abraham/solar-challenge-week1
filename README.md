# solar-challenge-week1

Setup Instructions

Clone repo: git clone https://github.com/<your-username>/solar-challenge-week1.git

Create virtual environment: python -m venv venv & to activate= venv/bin/activate

Install dependencies: pip install -r requirements.txt

Project Structure

├── notebooks/           # EDA & comparison notebooks

├── app/                 # Streamlit dashboard

├── .github/workflows/   # CI setup

├── requirements.txt

├── README.md

Tasks Summary

Task 1: Git setup, CI workflow, clean repo structure.

Task 2: EDA for Benin, Sierra Leone, Togo—cleaning, profiling, visual insights.

Bonus: Streamlit dashboard with country selector & GHI visualizations.

How to Run

Run notebooks for analysis.

Launch dashboard: streamlit run app/main.py
